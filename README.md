# Thiệp cưới online

Dự án thiệp cưới online có thể triển khai trực tiếp lên GitHub Pages.

## Cách chỉnh sửa
- Mở file `index.html`
- Sửa tên cô dâu, chú rể, thời gian, địa điểm
- Thay link Google Maps trong phần QR

## Deploy GitHub Pages
1. Tạo repo mới
2. Upload toàn bộ file
3. Vào Settings → Pages → chọn build from branch
4. Link sẽ tự tạo và có thể gửi cho khách
